# Book Store

## Overview

Book Store is a C# Windows Forms application designed to organize books into predefined categories and provide a simple, user-friendly browsing experience.

Users can explore different book sections and view details such as the title, author, and price of each book. The application also includes a suggestion feature that allows users to submit book recommendations, which are stored within the system for future consideration.

The project was developed to demonstrate Object-Oriented Programming (OOP) principles, including Inheritance and Indexers, while building an organized desktop application using Windows Forms.

## Features

- Browse books by category
- Display book information:
  - Title
  - Author
  - Price
- Organized category-based structure
- Submit and store book suggestions
- User-friendly Windows Forms interface
- Efficient book retrieval using Indexers
- Object-oriented design using Inheritance

## Technologies Used

- C#
- Windows Forms
- Object-Oriented Programming (OOP)

## OOP Concepts Implemented

### Inheritance
Inheritance was used to create relationships between classes and promote code reusability, making the application easier to maintain and extend.

### Indexers
Indexers were implemented to simplify access to books within each category, providing a convenient way to retrieve book information.

### Encapsulation
Class data and functionality were organized to ensure proper control over how information is accessed and modified.

## Suggestions System

The application includes a suggestion section where users can submit book recommendations.

Submitted suggestions are stored by the system and can be reviewed in future versions of the application. This feature provides a foundation for expanding the bookstore and improving user interaction.

## Project Structure

The application contains multiple predefined book categories. Each category stores a collection of books that users can browse. When a category is selected, the corresponding book details are displayed.

If a category does not contain any books, it remains empty until books are added.

## Objective

The main objective of this project is to apply C# programming skills and Object-Oriented Programming concepts by developing a desktop application that organizes books, manages user suggestions, and demonstrates the practical use of Inheritance and Indexers.
